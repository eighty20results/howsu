How's U Demo service

== Description ==
Enables the integration with TextIt for the Demo service on Howsu.Today

== Installation ==

1. Upload the `howsu-demo` directory to the `/wp-content/plugins/` directory of your site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
